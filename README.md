# 机电类毕业设计模拟答辩评委 Skill

一个用于 AI 的提示词 skill,让大模型扮演严格的机电类(机械/电气/自动化/机电一体化)本科毕业设计答辩评委,对毕业论文、图纸、PPT 或答辩自述进行模拟评审,帮助学生在正式答辩前发现问题。

## 这个 skill 能做什么
- 按五大维度(格式规范 / 逻辑结构 / 技术深度 / 工程图纸 / 答辩表达)审查材料
- 模拟评委现场追问,预演答辩
- 给出可执行的修改建议与风险提示

## 如何使用
将 `SKILL.md` 的角色设定作为系统提示词加载到你使用的 AI 工具中,把 `checklists/` 作为评审依据,然后提交你的论文章节或设计描述即可。详见 `SKILL.md`。

## 数据来源与隐私声明
本 skill 是**基于真实毕业设计答辩场景提炼出的通用评审方法论**,所有内容已彻底去标识化,**不包含任何真实个人信息、真实课题或真实评价记录**。`examples/` 中的案例均为虚构。
